# React Native To-Do App

A simple to-do app built using React Native.

## Features

- Add tasks to the to-do list.
- Delete tasks from the list.
- Save the tasks locally on the device.

## Installation

1. Clone the repository: `git clone https://github.com/ImXanax/ToDo.git`
2. Navigate to the project directory: `cd ToDo`
3. Install dependencies: `npm install`

## Usage

1. Run the app: `expo start`
2. The app will launch in your default emulator or on your connected device.
3. Add tasks by entering the task name and tapping the "-l||l-" button.
5. Delete tasks by tapping on them.

## Future Enhancements

- Implement the ability to set due dates and reminders for tasks.
- Add the ability to categorize tasks into different lists.
- Provide the option to sync tasks with cloud storage.
